Learning Rust Language

Este repositório é dedicado ao meu aprendizado da linguagem de programação Rust. Aqui você encontrará todo o material, 
exercícios e projetos desenvolvidos durante o curso "Learn Code with Rust", ministrado pelo Boris Paskhaver.
O objetivo é organizar e compartilhar meu progresso, facilitando a revisão dos conteúdos estudados e a troca de conhecimento com a comunidade.

1. Conteúdo

•	Estrutura do repositório organizada em seções, cada uma contendo exercícios e tarefas práticas.
•	Exemplos práticos e códigos comentados para facilitar o entendimento.
•	Experimentos e projetos simples para fixação dos conceitos de Rust.

2. Sobre o Curso

Estudo baseado no curso "Learn Code with Rust" da Udemy, que oferece um currículo dinâmico com vídeos, quizzes, 
exercícios escritos e desafios de programação, cobrindo desde fundamentos da linguagem até tópicos avançados como gerenciamento de memória, 
traits e organização de grandes projetos.

3. Tecnologias Utilizadas

•	Linguagem Rust
•	Ferramentas de desenvolvimento JetBrains RustRover
•	Git & GitHub para controle de versão e colaboração

4. Como usar este repositório

Clone este repositório para ter acesso a todos os exemplos e exercícios:

•	git clone https://github.com/JFicchini/learning-rust-language.git
Navegue pelas pastas das sessões para acompanhar o conteúdo estudado.

5. Contribuições

Este projeto é de estudo pessoal, mas sugestões, correções e discussões são sempre bem-vindas!
